`
Navigation:

    left-Arrow | h  : left
    up-Arrow | j    : down
    down-Arrow | k  : up
    right-Arrow | l : right
    
    shift + (left-Arrow | h)  : cycleChildren
    shift + (up-Arrow | j)    : jumpChild
    shift + (down-Arrow | k)  : jumpParent
    shift + (right-Arrow | l) : cycleParents

    m                    : moveRight
    shift + m            : moveLeft

    Enter | n            : jumpNextOccurrence
    shift + (Enter | n)  : jumpPrevOccurrence

    t (insideDefinition) : jumpNextTarget
    shift + t            : jumpPrevTarget

    g (insideDefinition) : jumpNextBlock
    shift + g            : jumpPrevBlock
    
    v         : jumpStart
    shift + v : jumpEnd
    
    u      : jumpBack


Features:

    Space:               : revealCursor
    
    Backspace | US-Slash : searchNode

    s (insideDefinition) : showTargetTree
    shift + s            : showCurrentNodeInfo
    shift + alt + s      : showKeyboardShortcuts

    b               : addBookmark
    shift + b       : revealBookmark
    shift + alt + b : removeBookmark

    c               : addComment
    shift + c       : removeComment
    shift + alt + c : resetAllComments

    x               : revealNextComment
    shift + x       : revealPrevComment

    r               : renameNode
    shift + r       : unnameNode
    shift + alt + r : resetAllNames

    shift + alt + f : foldAll
    shift + f       : foldBlocks
    f               : unfoldAll


Editor:

    tab      : toggleAllKeybinds

    q         : nextTCPH
    shift + q : prevTCPH
    
    shift + 1 : toggleNodeHighlighting
    shift + 2 : toggleVariableDecorating
    shift + 3 : toggleChildDecorating
    shift + 4 : toggleParentDecorating
    shift + 5 : toggleBookmarkDisplay
    shift + 6 : toggleCommentDisplay
    shift + 7 : toggleTargetTreeHover


Other:

    ctrl + f       : openFindDialog
    shift + esc    : closeToolWindow
    shift + scroll : scroll left / right
`
